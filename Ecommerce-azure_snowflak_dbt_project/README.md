## 📌 Project Overview

This project demonstrates an end-to-end modern Data Engineering and Analytics solution using Azure Databricks, ADLS Gen2, Snowflake, dbt, and Power BI.

The solution follows the Medallion Architecture (Bronze, Silver, Gold) to ingest, transform, curate, and analyze e-commerce data.

---

## 🏗️ Architecture

↓
Azure Databricks
(Bronze → Silver → Gold)

dbt
↓

## 🔄 Data Pipeline Flow

### Step 1: Data Ingestion

Raw CSV files are stored in ADLS Gen2.

Databricks accesses ADLS using:

* Managed Identity
* Storage Credential
* External Location
* External Volume
* Unity Catalog

---

### Step 2: Bronze Layer

Raw files are loaded into Bronze Delta Tables.

Tables:

* ecommerce_bronze.customers
* ecommerce_bronze.orders
* ecommerce_bronze.products
* ecommerce_bronze.payments

---

### Step 3: Silver Layer

Data cleansing and standardization:

* Remove duplicates
* Data type conversions
* Null handling
* Schema standardization

Tables:

* ecommerce_silver.customers
* ecommerce_silver.orders
* ecommerce_silver.products
* ecommerce_silver.payments

---

### Step 4: Gold Layer

Business-ready aggregated tables:

### customer_revenue

Revenue generated by each customer.

### product_sales

Total orders by product.

### daily_revenue

Daily revenue trends.

Tables:

* ecommerce_gold.customer_revenue
* ecommerce_gold.product_sales
* ecommerce_gold.daily_revenue

---

## ❄️ Snowflake Integration

Gold layer tables are loaded into Snowflake.

Database:
