# Airflow + Docker + Snowflake ETL Debugging Commands Cheat Sheet

## 1. Docker Commands

### Check Running Containers

```bash
docker ps
```

Purpose:

* Verify all containers are running.
* Check health status.
* Confirm port mappings.

Example:

```text
airflow-webserver
airflow-scheduler
airflow-worker
postgres
redis
```

---

### Check All Containers (Including Stopped)

```bash
docker ps -a
```

Purpose:

* Find containers that exited or crashed.

---

### View Container Logs

```bash
docker logs airflow-docker-airflow-webserver-1 --tail 100
```

Purpose:

* Diagnose Airflow UI issues.
* Check Gunicorn startup errors.
* View recent logs only.

---

### Follow Logs Live

```bash
docker logs -f airflow-docker-airflow-webserver-1
```

Purpose:

* Watch logs in real time.

---

### Restart Specific Container

```bash
docker restart airflow-docker-airflow-webserver-1
```

Purpose:

* Reload DAG changes.
* Restart failed service.

Also used:

```bash
docker restart airflow-docker-airflow-scheduler-1
docker restart airflow-docker-airflow-worker-1
```

---

### Enter Container Shell

```bash
docker exec -it airflow-docker-airflow-webserver-1 bash
```

Purpose:

* Inspect files.
* Run Airflow commands.
* Debug Python packages.

---

### Run Single Command Inside Container

```bash
docker exec -it airflow-docker-airflow-webserver-1 airflow db check
```

Purpose:

* Verify Airflow metadata database connection.

---

### View DAG File Inside Container

```bash
docker exec -it airflow-docker-airflow-webserver-1 head -50 /opt/airflow/dags/etl_postgres_to_snowflake.py
```

Purpose:

* Confirm deployed DAG version.

---

### Search Text Inside DAG

```bash
docker exec -it airflow-docker-airflow-webserver-1 grep -n "pd.read_json" /opt/airflow/dags/etl_postgres_to_snowflake.py
```

Purpose:

* Verify code changes reached container.

docker exec -it airflow-docker-airflow-webserver-1 airflow dags list-import-errors

docker exec -it airflow-docker-airflow-webserver-1 airflow dags list

Windows:

docker exec -it airflow-docker-airflow-webserver-1 airflow dags list
